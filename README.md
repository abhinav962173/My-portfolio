## 📌 About

This is my personal portfolio website — built from scratch using pure HTML and CSS.  
It serves as a central hub for everything I've built and learned so far.

## ✨ Features

- 🎨 Clean and minimal UI
- 📖 **Education** section — academic background
- 🛠️ **Skills** section — technologies I know
- 💼 **Projects** section — my GitHub projects showcased
- 📬 **Contact** section — reach out to me
- 🔗 External links to GitHub and LinkedIn profiles
- 📱 Responsive structure
